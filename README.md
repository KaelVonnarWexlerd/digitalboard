# Digital Board

靜態版 PDF 電子白板網站，依照 `design.md` 與 `skill.md` 規格建立。

## 使用方式

直接開啟 `index.html`，上傳 PDF 後即可進入全螢幕註記模式。

## 已完成

- PDF 上傳與 PDF.js 頁面渲染，載入後自動符合可用螢幕寬度
- 上方狀態列頁碼可直接輸入跳頁，狀態列可隱藏並由頂端把手叫回
- 每頁 PDF canvas 與 annotation canvas 分層
- 左右邊緣滑桿可展開 / 收合單一工具列，也可上下拖曳捲動 PDF；右側輔助功能整合到「更多」選單
- 筆、螢光筆、橡皮擦
- 鼠標工具拖曳平移頁面，搭配 Ctrl + 滾輪或右側按鈕縮放
- 套索工具支援矩形 / 徒手選取、四角與四邊縮放、旋轉、複製、剪下、貼上
- 36 色預設色、36 個自訂色、360 色調色盤與色碼輸入
- 清除目前頁註記
- 螢光筆固定濃度覆蓋，重複塗抹同色不會越來越深
- 線條、箭頭、平面圖形、立體簡圖、表格，支援圖形外框色、填色透明度、多邊形邊數與線條端點樣式
- Undo / Redo
- 矩形放大區域選取、四面八方都可平移與繼續書寫的純白無限白板放大頁、儲存標記點、刪除放大頁
- localStorage 本機暫存與 JSON 匯入 / 匯出
- 基本觸控與 Pointer Events 支援
